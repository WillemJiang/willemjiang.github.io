姜宁: 字节跳动开源办公室首席布道师，前华为开源管理中心技术专家，Apache 软件基金2022，2023，2024年度董事，Apache软件基金会孵化器导师， 前红帽软件首席软件工程师，Apache 本地北京社群（ALC Beijing）发起人，有十余年企业级开源中间件开发经验，有丰富的 Java 开发和使用经验，函数式编程爱好者。从2006年开始一直从事 Apache 软基金会开源中间件项目的开发工作，先后参与 Apache CXF， Apache Camel，Apache ServiceMix，Apache ServiceComb 的开发。对微服务架构，WebServices，Enterprise Integration Pattern，SOA， OSGi有比较深入的研究。
